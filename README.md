# prrr

This is a simple tool to help you manage your pull requests.

## Requirements

This project requires and OpenAPI token to exist in your environment variables.

```shell
OPENAI_API_KEY=your_api_key
``` 

## Installation

```bash
pip install prrr
```

## Usage

```shell
prrr pr gen
```
